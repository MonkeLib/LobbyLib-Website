LobbyLib

LobbyLib is the most easy and simple yet to use powerful lobby plugin available.

Easy to use

Able to easily access all the beyond by just using our commands.

Highly customizable

You can edit the config files the way you would like to enhance the experience for your players.

Optimal for servers

We are a light weight plugin which means that mostly any server can easily run smoothly with LobbyLib.
